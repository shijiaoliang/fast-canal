# fast-canal

fast-canal is fast and pretty used for sync mysql data to any where.

## Quick Start

#### Create a run directory
    mkdir fast-canal
    cd fast-canal
    vim config.toml

#### Download release
    fast-canal/release/fast-canal

#### Run the backend
    nohup ./fast-canal>/dev/null &
