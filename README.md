# fast-canal

fast-canal is fast and pretty used for sync mysql data to any where.

## Quick Start

#### From MySql Latest Position
    mysql> flush logs;

#### Clone & Run
    git clone https://github.com/shijiaoliang/fast-canal.git
    cd fast-canal
    vim config.toml
    go mod tidy
    go run main.go
